# Images Folder

## Required Images

To complete your portfolio website, please add the following images to this folder:

### 1. **project1.jpg**
- Screenshot or thumbnail of your first project
- Recommended size: 800x600px or similar aspect ratio (4:3 or 16:9)
- Format: JPG, PNG, or WebP

### 2. **project2.jpg**
- Screenshot or thumbnail of your second project
- Recommended size: 800x600px or similar aspect ratio (4:3 or 16:9)
- Format: JPG, PNG, or WebP

### Optional Images

- **profile.jpg** - Your profile photo (if you want to add it to the About section)
- **favicon.ico** - Website favicon for browser tab

## Image Optimization Tips

1. Compress images to reduce file size (use tools like TinyPNG or ImageOptim)
2. Use appropriate formats: JPG for photos, PNG for graphics with transparency
3. Consider using WebP format for better compression (with JPG/PNG fallbacks)
4. Keep file sizes under 500KB for faster loading

## Placeholder Images

If you don't have images yet, you can use placeholder services:
- https://placehold.co/800x600
- https://picsum.photos/800/600
- https://via.placeholder.com/800x600

Simply replace the image URLs in `index.html` with these placeholders temporarily.
